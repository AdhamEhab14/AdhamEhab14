## Projects

### [CoTune Remote Control Laboratory](https://github.com/AdhamEhab14/CoTune-Remote-Control-Lab)
Graduation project: a remote-access control systems lab across 4 live testbeds on ESP32/FreeRTOS/AWS IoT Core, streaming telemetry and video at sub-1s latency. State-feedback LQR control and multi-rate Kalman filtering, with an Augmented EKF digital twin reaching 2.6× the estimation accuracy of the nominal model. [Website](https://main.d35sofljaoe99e.amplifyapp.com/) · [Demo](https://www.youtube.com/watch?v=JONjcuDR5eo)

### [Smart Beehive IoT Node & Telemetry Dashboard](https://github.com/AdhamEhab14/Smart-Beehive-IoT-Node)
Dual-priority FreeRTOS architecture on ESP32 that separates a 500ms local display loop from a 5s MQTT-over-TLS telemetry pipeline, with a Flutter + Firebase dashboard for live monitoring and remote actuator control.

### [Low-Level Peripheral Drivers — PIC18 & STM32](https://github.com/AdhamEhab14/PIC18-STM32-Peripheral-Drivers)
Register-level, bare-metal peripheral drivers for the PIC18F46K20 and STM32F103RB, reused across 5+ projects.
